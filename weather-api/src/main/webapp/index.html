1. sequential access by APIs:

  • request the latest weather data, replace the token http://localhost:8080/api/weather?token=replaceme
  • request weather data by range, replace the token http://localhost:8080/api/weather?start=xxx&end=XXX&token=replaceme

2. Access the application management Endpoints:

  • http://localhost:7002/health
  • http://localhost:7002/info
  • http://localhost:7002/dump
  • http://localhost:7002/metrics
  • http://localhost:7002/env
  • http://localhost:7002/mappings
  • shutdown(disable by default, POST method)

3. JMX: through Restful api with jolokia:

  • Tomcat MBean

4. JAVA Simon: a simple monitoring API for Application :
